Patterns

The myuw-help component loosely follows guidelines set by Material Design's guidelinesopen_in_new.
    Do
  • Keep the content of the help dialog simple (no walls of text, fewer than 10 links)
  • Provide help specific to your app (e.g. Accessibility assistance) as well as institutional help, when possible (e.g. Help Desk contact information)
  • Keep the help dialog content the same throughout the application (with the exception of additional contextual information)
  • Optionally add contextual help content (i.e. A task-oriented page may have additional information in the help dialog when the user is viewing that page)

    • Don't
  • Provide different help dialog content on different pages (only include additional contextual info)

How to use

Refer to the readmeopen_in_new for detailed information.


<myuw-help
  slot="myuw-help"
  myuw-help-title=""
  show-button
  show-default-content>
  <div slot="myuw-help-content"></div>
</myuw-help>

Attributes

  • myuw-help-title: The title to display at the top of the help dialog
  • show-button: Include this attribute if you want the help icon button to appear the the top bar. If you want to trigger the dialog some other way, you're free to omit this attribute.
  • show-default-content: Include this attribute if you want to include the default UW-Madison-flavored help links. At this time we do not recommend showing the default content, as it is still a work in progress.

Slots

  • myuw-help-content: Use this slot to insert your own content into the dialog (with whatever markup and styling you want).

Custom Events

If you aren't using the top bar button (via the show-button attribute), you can fire the custom event "show-myuw-help" event on the when you want the dialog to display (e.g. when your "help" button is clicked). If you need to control the exact positioning of the dialog, you can fire the "set-myuw-help-position custom event. 


// Show the dialog
function showHelpDialog() {
  var event = new Event('show-myuw-help');
  document.dispatchEvent(event);
}

// Set position (defaults to horizontally center, vertically centered in top 1/2)
function setDialogPosition() {
  var event = new CustomEvent('set-myuw-help-position', {
    detail: {
      position: {       // Include a "position" object only if you want to control the
        top: '100px',   // exact positioning of the dialog
        left: '100px',
      }
    }
  });

  document.dispatchEvent(event);
}
